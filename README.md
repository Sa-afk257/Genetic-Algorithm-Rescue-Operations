# Genetic Algorithm Rescue Operations

## Overview

This project applies Genetic Algorithms to optimize rescue robot operations in collapsed building environments.

The system models disaster zones as 3D grids and generates efficient robot paths to maximize survivor coverage while minimizing travel distance, operation time, and risk exposure.

The implementation utilizes Linux IPC techniques and multiprocessing for parallel evaluation of candidate solutions.

## Features

* 3D Rescue Environment Modeling
* Genetic Algorithm Optimization
* Chromosome-Based Path Representation
* Tournament Selection
* Crossover Operations
* Mutation Operations
* Elitism Support
* Survivor Coverage Optimization
* Collision Detection
* Multi-Robot Coordination

## Technologies Used
